Nuzi Raidtools v1.0.0

- first full release for nuzi-raidtools
- fixes the load-time helper ordering error that could prevent the addon from starting
- keeps settings and list data under `.data` so updates are less likely to wipe saved files
- migrates the legacy expedition whitelist into `.data` automatically
- cleans up the stock raid info screen to focus on `Name / Class / GS`
- includes auto-invite tools, whitelist and blacklist management, auto role support, and give-lead controls
- adds a README with install notes and a quick how-to
